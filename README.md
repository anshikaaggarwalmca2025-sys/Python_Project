# Financial Fraud Detection System (100k Records)

## 📊 Project Overview
This project involves the analysis of a large-scale financial dataset containing **100,000 transactions**. Using **Python** and **Pandas**, I identified patterns of fraudulent activity and visualized the results to provide actionable insights.

## 🚀 Key Features
* **Big Data Handling:** Successfully processed 100,000 records using the Pandas library.
* **Fraud Detection:** Grouped and filtered data to identify high-risk locations and device types.
* **Visual Insights:** Developed clear visualizations (Pie Charts & Bar Graphs) using Matplotlib.
* **Scalability:** Demonstrated how Python outperforms manual tools (like basic Excel) for large datasets.

## 🛠️ Tech Stack
* **Language:** Python
* **Libraries:** Pandas, Matplotlib, NumPy
* **Platform:** Google Colab
